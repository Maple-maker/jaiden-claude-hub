Give Claude a memory so you stop repeating yourself

A CLAUDE.md file is a plain document Claude reads at the start of every session — your role, preferences, and rules. Set it once. Every conversation starts already briefed.

Read guide Start here

Teach Claude the why, not just the what

Prompts that explain the reason behind a rule generalize far better than prompts that list examples. One shift in how you write instructions changes everything Claude produces for you.

Read guide Prompting 101

Stop re-typing. Let Claude run the whole task.

Instead of feeding it one instruction at a time, you set up a simple loop: gather, do, check, repeat — and Claude works through a real task on its own. The single biggest shift in how to use AI.

Read guide Loops

Connect Claude to the apps you already use

Claude can only help with what it can see. A connector links it to your real tools — calendar, email, files, notes — so it acts on your actual stuff instead of guessing. No code to set up.

Read guide Connectors

Put a whole team of AIs on one job

One AI doing everything gets overwhelmed and sloppy. Instead you split the work — each helper gets one clear job, a lead pulls it together. It's how you tackle big things without the quality slipping.
